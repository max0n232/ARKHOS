# Content Strategy Rubric

## Dimensions

### 1. Audience definition (Определение аудитории)
- 1: No audience segmentation
- 2: Generic demographics
- 3: Personas with pain points
- 4: Data-backed personas with platform-specific behavior
- 5: Validated personas with content consumption patterns per platform (IG, FB, TikTok)

### 2. Content-platform fit (Соответствие платформе)
- 1: Same content everywhere
- 2: Acknowledges platforms differ
- 3: Platform-specific formats described
- 4: Specific format templates per platform with examples (Reels dimensions, caption length, hashtag strategy)
- 5: Algorithm-aware strategy with posting schedules, engagement patterns, cross-posting rules

### 3. Production pipeline (Пайплайн производства)
- 1: No production process
- 2: Idea → post (manual)
- 3: Content calendar + basic workflow
- 4: Automated pipeline: ideation → creation → review → scheduling → analytics (n8n integration points)
- 5: Full pipeline with A/B testing, performance feedback loops, repurposing strategy, team roles

### 4. Measurement framework (Система метрик)
- 1: No KPIs
- 2: Vanity metrics only (likes, followers)
- 3: Engagement + reach metrics
- 4: Funnel-aligned metrics: awareness → consideration → conversion with tracking setup
- 5: Attribution model with UTM strategy, Meta Pixel events, and automated reporting

### 5. Differentiation (Дифференциация)
- 1: Generic kitchen content
- 2: Mentions brand positioning
- 3: Clear USP with content pillars
- 4: Competitive content audit + unique angle with proof points
- 5: Blue ocean content strategy with untapped format/topic combinations in Estonian market
