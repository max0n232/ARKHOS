# SEO/GEO Rubric

## Dimensions

### 1. Technical completeness (Техническая полнота)
- 1: Missing critical elements (no mention of Core Web Vitals, no mobile considerations)
- 2: Covers basics but missing modern requirements (Schema.org, hreflang, structured data)
- 3: Good coverage but gaps in implementation details
- 4: Comprehensive with specific implementation steps and priorities
- 5: Enterprise-grade plan with monitoring, fallbacks, and automation

### 2. Market specificity (Специфика рынка)
- 1: Generic advice, could apply to any country
- 2: Mentions Estonia but no specific data or local competitors
- 3: References Estonian market but lacks concrete competitor analysis
- 4: Includes Estonian search behavior, local competitors, language-specific keyword data
- 5: Deep local insights: Estonian vs Russian search patterns, seasonal trends, local directories, regional Google features

### 3. Actionability (Конкретность действий)
- 1: Vague recommendations ("improve content", "optimize pages")
- 2: Named tasks but no prioritization or timeline
- 3: Prioritized tasks with rough timeline
- 4: Sprint-level breakdown with owners, tools, and measurable KPIs
- 5: Ready-to-execute with n8n automation hooks, WP-CLI commands, and monitoring dashboards

### 4. Competitive analysis (Конкурентный анализ)
- 1: No competitor mention
- 2: Names competitors without analysis
- 3: Basic comparison (traffic estimates, keyword overlap)
- 4: Gap analysis with specific opportunities and differentiation strategy
- 5: SERP feature analysis, content gap mapping, backlink opportunity matrix

### 5. AI-readiness / GEO (AI-оптимизация)
- 1: No mention of AI search (Google AI Overviews, Bing Copilot, Perplexity)
- 2: Mentions AI search exists
- 3: Basic GEO recommendations (structured data, clear answers)
- 4: Specific GEO strategy: LLMs.txt, entity markup, FAQ optimization for AI extraction
- 5: Full GEO audit with testing methodology, AI citation tracking, and optimization pipeline
