### Script Structure

**Tutorial (8-12 min):**
```
[0:00-0:15] Hook + problem
[0:15-0:45] Intro + what they'll learn
[0:45-1:00] Subscribe CTA
[1:00-8:00] Main content (3-5 steps)
[8:00-9:00] Recap + tips
[9:00-9:30] End screen + next video
```

**Shorts (45-60s):**
```

[10-45s] Solution (3 quick steps)
[45-60s] CTA + follow
```

## Content Pillars Template

For any channel, define 3-5 pillars:

**Example (Kitchen Design Channel):**
1. **Before/After** - transformations
2. **Tips & Hacks** - quick wins
3. **Product Reviews** - honest opinions
4. **Industry Insights** - trends, news
5. **Q&A** - audience questions

Rotation: Don't repeat same pillar 2x in a row
