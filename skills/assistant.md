### Productivity Techniques

**Pomodoro:**
```
25 min work ‚Üí 5 min break
After 4 cycles ‚Üí 15-30 min break

Use for: Deep work, coding, writing
Avoid: Meetings, interruptions
```

**Time Blocking:**
```
9:00-11:00   Deep Work (no interruptions)
11:00-12:00  Emails & Admin
12:00-13:00  Lunch
13:00-15:00  Meetings
15:00-17:00  Creative Work
17:00-18:00  Planning & Review

Color code by category
```

**2-Minute Rule:**
```
If task takes <2 min ‚Üí Do it NOW
Don't add to list, just complete
```

### Energy Management

### Decision Making

**Decision Framework:**
```
For each choice:

1. What's the goal?
2. What are options? (3-5 max)
3. Criteria for success? (rank 1-5)
4. Score each option
5. Pick highest score
6. Set deadline to decide
7. Execute & don't second-guess

For big decisions: Sleep on it
For small decisions: 2-minute rule
```

**Avoiding Analysis Paralysis:**
```
If stuck >30 min:
1. Flip a coin (seriously)
2. Which feels right?
3. What would you advise a friend?
4. What's reversible? (then pick fast)
```
