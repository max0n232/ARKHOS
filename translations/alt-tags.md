## Как применить в TranslatePress

1. Открыть страницу в режиме редактирования TranslatePress
2. Навести на изображение
3. В появившемся окне найти поле "Alt text"
4. Ввести перевод из таблицы выше
5. Сохранить
6. Повторить для каждого языка
